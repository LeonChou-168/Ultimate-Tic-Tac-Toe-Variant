# Code of Conduct

## Our Standard

This project aims to be a respectful, focused, and collaborative space.

Examples of positive behavior include:

- Giving constructive, specific feedback
- Respecting different skill levels and backgrounds
- Keeping technical disagreement professional and evidence-based
- Reporting bugs and proposing improvements in good faith

Examples of unacceptable behavior include:

- Harassment, personal attacks, or intimidation
- Discriminatory or hateful language
- Deliberate disruption of discussion or project workflow
- Publishing private information without permission

## Scope

This Code of Conduct applies within project discussions, issues, pull requests, and other community spaces tied to this repository.

## Enforcement

Project maintainers may remove, edit, or reject comments, issues, code, or other contributions that violate this Code of Conduct.

## Reporting

If you encounter unacceptable behavior, please open a private security-style report if possible or contact the maintainer through the repository owner account.

## Notes

This is a lightweight repository-level conduct policy intended to keep collaboration clear and respectful.
